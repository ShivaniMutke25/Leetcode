# Append Characters to String to Make Subsequence - LeetCode #2486

**Link:** https://leetcode.com/problems/append-characters-to-string-to-make-subsequence/

## Problem Description
You are given two strings `s` and `t`. String `t` is generated by random shuffling string `s` and then adding one more letter at a random position.

Return the minimum number of characters from `s` that must be appended to `t` to make `t` a subsequence of `s`.

**Example:**
- Input: s = "abcde", t = "aabcbb"
- Output: 3
- Explanation: The subsequence "abcde" can be obtained by adding 'a', 'b', and 'c' to "abcde".

**Constraints:**
- 1 <= s.length, t.length <= 10^5
- s and t consist of lowercase English letters.
